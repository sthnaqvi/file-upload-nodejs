# File upload nodejs

### Dependencies
- express
- ejs
- fs
- multer
- localtunnel



### Start
```
> git clone https://github.com/sthnaqvi/file-upload-nodejs.git
> cd file-upload-nodejs
> npm install
> npm start
> npm run lt-server (start server with localtunnel url)
